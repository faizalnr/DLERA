# DLERA
**Dynamic Lightweight RSA-Based Encryption for Secure Software-Defined Networks**

This code is part of our new project titled "DLERSA: Dynamic Lightweight RSA-Based Encryption for Secure Software-Defined Networks" which is ongoing review process in reputed journal.

Run the Python code DLERSA-SingleData for single image processing and DLERSA_Dataset.py for Dataset processing. 
Dataset file added as "DLERSA_Samples.xlsx". 
Results will be saved in output folder.
